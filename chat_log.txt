DATABASE SETUP (Pending — to be done by team)
═══════════════════════════════════════════════════════════
  1. Install PostgreSQL: https://www.postgresql.org/download/windows/
  2. psql -U postgres -c "CREATE DATABASE csmip_db;"
  3. psql -U postgres -c "CREATE USER csmip_user WITH PASSWORD 'csmip_password';"
  4. psql -U postgres -c "GRANT ALL PRIVILEGES ON DATABASE csmip_db TO csmip_user;"
  5. cd backend && npx prisma db push
  6. cd backend && node prisma/seed.js
  (Once done, mock data is auto-replaced by real DB data — no code changes needed)
