    TO FIX THE DATABASE (required for full functionality):
    1. Install PostgreSQL from https://www.postgresql.org/download/windows/
    2. Run: psql -U postgres -c "CREATE DATABASE csmip_db;"
    3. Run: psql -U postgres -c "CREATE USER csmip_user WITH PASSWORD 'csmip_password';"
    4. Run: psql -U postgres -c "GRANT ALL PRIVILEGES ON DATABASE csmip_db TO csmip_user;"
    5. In /backend run: npx prisma db push
    6. In /backend run: node prisma/seed.js
